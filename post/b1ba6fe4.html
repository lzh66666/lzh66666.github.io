安装DevC++ 5.11和MySQL

网上教程很多，自己去找吧

安装MySQL.Devpak

  1. 在DevC++的Tools里可以找到Package Manager点击
  2. 安装成功

    环境配置

  3. 系统环境添加libmySQL.dll
    找到如下目录　C:\Windows\System32

    添加 libmySQL.dll 文件，该文件在mysql文件的lib目录中

开始操作

1.新建一个c++项目

2.DevCpp顶部工具栏 -> 项目 -> 项目属性 -> 参数

  • 在C complier里输入-Wall
  • 在C++ complier里输入-W
  • Linker里->Add library or object，（找到DevC++安装目录/lib/libmysql.a）
    • 加入libmysql.a该库
    • 加入 -lmysql
  • DevCpp顶部工具栏 -> Project -> Project Options… -> Directories
    • Library Directories：（安装的MySQL的目录）/lib
    • Include Directories：（安装的MySQL的目录）/include
